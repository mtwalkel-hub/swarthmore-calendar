How to Subscribe

  1. Click "Add to Google Calendar" above, or:
  2. In Google Calendar, click the + next to "Other calendars"
  3. Select "From URL"
  4. Paste this URL:
    https://mtwalkel-hub.github.io/swarthmore-calendar/swarthmore_events.ics
  5. Click "Add calendar" - events will sync automatically!

Events sourced from The Dash. Calendar refreshes daily via GitHub Actions.
